# Traffic-Control-System
To Access the main Application that shows the working of Traffic Control system , Follow these Steps :

1. On the right side , there is a heading named RELEASES below this heading there is link with name Traffic_Simulation Click on it.
2. After clicking on it a new window will appear in which 3 to 4 files will be visible.
3. From this section locate the file named TrafficSystem_Pro.exe and then click it .
4. It will start downloading after it finishes open the app , or run the application on your device
